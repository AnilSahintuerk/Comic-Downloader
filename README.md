# Comic-Downloader

## Description

With this project you can download a comic from webpages. It will create a folder for a single chapter and scrape all images according to that comic. After downloading a chapter, all images in a folder get converted in a single pdf file, with all images in order.

## Prerequisites

- Python 3.6
- Python Libraries:
    - requests
    - bs4
    - os 
    - re (optional) 

## Installation

1. Clone the repository locally on your machine
2. Open terminal and change directory to the local repository
3. Run in the terminal: *python3.6 script.py*

## How to use this Project?

Add your own extensions (See docs/extension (!Add Link)) to download your the comic you want from a specific website. It is necessary to have an understanding in website scraping and how to get a link from a html tag.
